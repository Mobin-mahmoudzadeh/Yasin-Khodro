# 🚗 Yasin Khodro

Yasin Khodro is a responsive Persian car rental website built with HTML, CSS, and JavaScript.

The project features a modern car rental interface with trending vehicles, rental listings, team members, customer reviews, a newsletter section, and a responsive mobile navigation menu.

## ✨ Features

- Fully responsive design
- Persian and RTL-friendly interface
- Modern navigation bar
- Responsive mobile menu
- Trending cars section
- Car rental listings
- Team members section
- Customer reviews
- Newsletter subscription section
- Footer with contact and social information
- Scroll reveal animations
- Custom responsive breakpoints for different screen sizes

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Boxicons
- ScrollReveal
